Twitter Spaces CC Expander

Enlarge the caption area of Twitter spaces, only works on Twitter (browser page) for pc.

  1. Copy the script below.
  2. Go to the Twitter page.
  3. Open a Twitter spaces, and make sure there is already have subtitle shows.

    4. (This script will not work if automatic subtitle translation (cc) is not enabled, or if the translated subtitles are not yet available.)

  4. Press "F12" to open the "Developer Tools".
  5. Find the "console" in Developer Tools.
  6. Paste the script and click "Enter" to execute it.
  7. Now the subtitles will be enlarged and displayed on the side.
